# 00 Welcome

Welcome to this Object-Oriented Programming LearnPack in Python.

Here you will learn how to model real-world scenarios with classes and objects, step by step.

## What we will do

This step, 00-Welcome, is introduction only.

Starting in the next module, you will work through 11 progressive practices to master:

- Classes and instances
- Attributes and methods
- Encapsulation
- Inheritance
- Polymorphism
- Applied abstraction
- Composition and object collections

## How to work on each exercise

1. Read the module README.md.
2. Implement your solution in app.py.
3. Run Run to check behavior.
4. Run Test until all tests pass.
5. Click Next to continue.

## Learning path

- 01-03: class fundamentals, state, and instance methods
- 04-06: encapsulation, validation, and special methods
- 07-09: class attributes, composition, and object collections
- 10-11: inheritance, polymorphism, and abstraction

## Final goal

By the end, you will be able to design clear classes, keep object state consistent, and apply OOP pillars to real problems.

When you are ready, move on to 01-Create-First-Class.
