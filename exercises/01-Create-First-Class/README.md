# 01 Create your first class

## Goal

In this first exercise, you will practice the foundation of OOP: defining a class and creating an object from it.

## Instructions

Work in `app.py` and complete exactly the following:

1. Create a class named `Person`.
2. Create an instance named `student` using that class.

In this module, do not add attributes or methods yet.

## Success criteria

The exercise is correct when:

- The `Person` class exists.
- `student` is an instance of `Person`.

## Expected example

```python
class Person:
    pass

student = Person()
```
